# SmartHome

智能家居&报警系统

## 硬件设备

- MCU：STM32F103C8T6
- 联网模块：ESP8266-01S
- 温湿度传感器：DHT11
- 光照传感器：BH1750
- 无线射频模块：433M
- 其他：OLED、BEEP、LED、KEY*2、USB转串口模块(CP2102)

### 实际接线图

采用面包板接线，肯定没有采用PCB板稳定。这样导致在运行中可能出现问题，例如：ESP8266-01S可能因为接线不稳定导致运行错误。
